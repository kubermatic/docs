## Supported Operating Systems

The following operating systems are supported:

* Ubuntu 20.04 (Focal)
* Ubuntu 22.04 (Jammy Jellyfish)
* Ubuntu 24.04 (Noble Numbat)
* Rocky Linux 8
* RHEL 8.0, 8.1, 8.2, 8.3, 8.4
* Flatcar
* Amazon Linux 2
