## kubelb tunnel get

Get a tunnel

### Synopsis

Retrieve a tunnel by name and output it's complete YAML specification.


```
kubelb tunnel get NAME [flags]
```

### Examples

```
kubelb tunnel get my-app --tenant=mytenant --kubeconfig=./kubeconfig
```
