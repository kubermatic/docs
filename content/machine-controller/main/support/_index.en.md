## Getting Help

If you need help with machine-controller:

1. **Check the documentation** - Browse through the tutorials, references, and troubleshooting guides
2. **Search existing issues** - Check [GitHub Issues](https://github.com/kubermatic/machine-controller/issues) for similar problems
3. **Join the community** - Ask questions in the [#kubermatic channel](https://kubermatic.slack.com/messages/kubermatic) on [Kubermatic Slack](http://slack.kubermatic.io/)
4. **File an issue** - If you've found a bug, please [file an issue](https://github.com/kubermatic/machine-controller/issues/new)
